# Instrucciones de Contribución

Gracias por considerar contribuir a autoBixpe. 

## Cómo reportar bugs

1. Abre un issue con título descriptivo
2. Incluye pasos para reproducir el problema
3. Incluye logs relevantes
4. Menciona tu versión de Python, Docker y SO

## Cómo sugerir mejoras

1. Usa el título claro que describa la mejora
2. Explica el caso de uso actual vs el deseado
3. Lista ejemplos de proyectos similares si aplica

## Pull Requests

1. Fork el repositorio
2. Crea una rama (`git checkout -b feature/tu-mejora`)
3. Commit tus cambios (`git commit -m 'Agrega tu mejora'`)
4. Push a la rama (`git push origin feature/tu-mejora`)
5. Abre un Pull Request

## Estándares de código

- Usa PEP 8 para Python
- Comenta tu código
- Actualiza el README si es necesario
- Prueba en Docker antes de hacer PR

## Licencia

Al contribuir, aceptas que tu código será licenciado bajo MIT.
